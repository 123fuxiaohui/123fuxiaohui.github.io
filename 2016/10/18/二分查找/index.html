什么是二分查找?

二分查找的基本思想是将n个元素分成个数大致相同的两半，取a[n/2]与x作比较，如果x=a[n/2],则找到x,算法终止，如果想x<a[n/2]
则在数组a的左半部搜索x,如果x>a[n/2],则只在数组a的右半部继续搜索x.

求任意一个position使得datas[position]==num，找不到返回-1

public int BinarySearch(int[] array,int l,int r,int num){
    int mid,position=-1;
    while(true){
        if(l>r){
            break;
        }
        mid=l+(r-l)/2;
        if(num>array[mid]){
            l=mid+1;
        }else if(num==array[mid]){
            position=mid;
            break;
        }else{
            r=mid-1;
        }
    }
    return position;
}

求最大(最小)的position使得array[position]==num,找不到返回-1

public int BinarySearch(int[] array,int l,int r,int num){
    int mid,position=-1;
    while(true){
        if(l==r-l){
            break;
        }
        mid=l+(r-l)/2;
        if(num>array[mid]){
            l=mid+1;
        }else if(num==array[mid]){
            l=mid;
        }else{
            r=mid-1;
        }
    }
    if(array[r]==num){
        position=r;
    }else if(array[l]==num){
        position=l;
    }
    return position;
}
